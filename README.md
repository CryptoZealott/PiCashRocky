# PiCash

A lightweight docker image running many "passive" income applications, built for a Raspberry Pi.

Currenlty installed programs are:
- Honeygain
- EarnApp 
- IPRoyal Pawns 
- PacketStream
- Peer2Profit

64bit Raspberry Pi's are supported (4, z2w, 3b+)
